## 🔧 Настройки

### Безопасность

```python
# bot/telegram_bot.py
if message.from_user.id != self.admin_id:
    return  # Только admin может использовать команды
```

**Как получить admin_id:**
1. Откройте @userinfobot в Telegram
2. Нажмите Start
3. Скопируйте ID (например: 123456789)
4. Вставьте в `.env`:
   ```
   TELEGRAM_ADMIN_ID=123456789
   ```

### Cooldown

```python
# Защита от спама
self._cooldown_seconds = 5  # 5 секунд между алертами
```

## 🛠️ Troubleshooting

### Команды не работают

**Проблема:** Бот не отвечает

**Решение:**
1. Проверьте `/start` - бот активен?
2. Проверьте токен в `.env`
3. Проверьте admin_id
4. Перезапустите бота

### `/status` показывает 0

**Проблема:** Нет данных

**Решение:**
1. Запустите скринер: `python main.py`
2. Подождите алертов
3. Проверьте логи

### `/logs` пустой

**Проблема:** Нет записей в логах

**Решение:**
1. Проверьте `logs/alerts.log`
2. Запустите скринер
3. Подождите алертов

### `/set_threshold` не работает

**Проблема:** Ошибка установки

**Решение:**
1. Проверьте формат: `/set_threshold 500000`
2. Минимум: $10,000
3. Проверьте права admin
