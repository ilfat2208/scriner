## 💡 Интерпретация сигналов

### 🟢 OI Increase + Price ↑
**Сигнал:** Бычий тренд
- Приток новых денег
- Уверенность покупателей
- **Действие:** Лонг

### 🔴 OI Decrease + Price ↓
**Сигнал:** Медвежий тренд
- Закрытие позиций
- Паника/фиксация
- **Действие:** Шорт или выход

### 🟢 OI Increase + Price ↓
**Сигнал:** Медвежий дивергенция
- Агрессивные шорты
- Давление продавцов
- **Действие:** Осторожно, возможен шорт

### 🔴 OI Decrease + Price ↑
**Сигнал:** Бычий сквиз
- Закрытие шортов
- Короткий сквиз
- **Действие:** Быстрый скальп

## 🔧 Настройки

### Частота проверок

```python

if current_time - last_oi_check > 60:  # Каждые 60 секунд
    await self._check_open_interest_for_all_pairs()
```

### Порог срабатывания

```python

    oi_change_threshold=3.0,  # 3% изменение
    window_minutes=5,         # 5 минут окно

```

### Cooldown (защита от спама)

```python
# bot/telegram_bot.py
if delta < self._cooldown_seconds:
    return  # Пропуск частых алертов
```

## 🎯 Рекомендации для пользователей

### Как торговать по OI сигналам:

1. **Получите уведомление:**
   ```
   📈 BTC/USDT OI increased by 5.23%
   ```

2. **Проверьте контекст:**
   - Нажмите 📊 TradingView
   - Проверьте тренд
   - Посмотрите уровни

3. **Проверьте Funding:**
   - Нажмите 📊 Funding
   - Отрицательный funding = бычий сигнал
   - Положительный funding = медвежий

4. **Примите решение:**
   - OI ↑ + Price ↑ = Лонг
   - OI ↓ + Price ↓ = Шорт
   - OI ↑ + Price ↓ = Осторожно

5. **Установите стоп-лосс:**
   - Ниже последнего минимума
   - 2-3% от входа

## 🔍 Troubleshooting

### Уведомления не приходят

**Проблема:** Бот не отправляет сообщения

**Решение:**
1. Проверьте токен в `.env`
2. Проверьте admin_id
3. Запустите `/start` в боте

### Слишком много уведомлений

**Проблема:** Спам алертами

**Решение:**
1. Увеличьте `oi_change_threshold` до 5%
2. Увеличьте `cooldown_seconds` до 10
3. Фильтруйте пары (только топ-50)

### Кнопки не работают

**Проблема:** Ссылки не открываются

**Решение:**
1. Проверьте формат URL
2. Используйте https://
3. Обновите Telegram
